Importando a Base de dados Relacional criada acima para dentro do HDFS com o SQOOP.

o SQOOP executa um MAP/REDUCE.

Se fosse fora do container o comando seria.
sqoop import --connect jdbc:mysql://localhost:3306/testedb --username root --password secret --table empregados --m 1
